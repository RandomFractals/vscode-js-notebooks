# Dev Build

```bash
$ git clone https://github.com/RandomFractals/vscode-js-notebooks
$ cd vscode-js-notebooks
$ npm install
$ code-insiders .
```
`F5` to launch JS Notebooks 📚 extension VSCode debug session.
